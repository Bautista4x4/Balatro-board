📡 Sincronización Paso a Paso

Firebase Firestore · Gratis · En tiempo real

1

Crea un proyecto Firebase gratuito

Ve a console.firebase.google.com. Haz clic en "Añadir proyecto", ponle un nombre, desactiva Google Analytics y presiona "Crear proyecto".

2

Añade una aplicación Web

En la pantalla de tu nuevo proyecto, haz clic en el icono Web (</>). Ponle un nombre a la app, haz clic en "Registrar app" y copia el código que aparece dentro de firebaseConfig = { ... }.

3

Habilita Firestore Database

En el menú lateral izquierdo, ve a "Compilación" -> "Firestore Database". Haz clic en "Crear base de datos", selecciona la ubicación más cercana y empieza en Modo Producción.

4

Actualiza las reglas de seguridad

Dentro de Firestore, ve a la pestaña "Reglas", borra todo y pega lo siguiente, luego pulsa Publicar:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
5

Habilita autenticación anónima

En el menú izquierdo, ve a "Authentication" -> "Comenzar". En la pestaña "Sign-in method", habilita el proveedor "Anónimo" y guarda.

6

Pega la configuración en la pestaña "Config"

Pega el código copiado en el paso 2 en la pestaña "Config" de esta ventana. Usa la misma semilla en otros dispositivos y ¡listo!
