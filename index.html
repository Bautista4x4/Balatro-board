📡 Sincronización entre Dispositivos

Firebase Firestore · Gratis · Tiempo real

1

Crea un proyecto Firebase gratuito

Ve a console.firebase.google.com → "Añadir proyecto" → desactiva Google Analytics → Crear.

2

Añade una app Web

En tu proyecto → ⚙ Configuración del proyecto → "Tus apps" → icono </> → copia el bloque firebaseConfig.

3

Activa Firestore

Menú lateral → "Firestore Database" → "Crear base de datos" → modo Producción → elige región → Listo.

4

Abre las reglas de seguridad

Firestore → pestaña "Reglas" → reemplaza todo con esto y haz clic en "Publicar":

rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
5

Pega tu config en la pestaña "Pegar config"

Repite en cada dispositivo. Misma semilla = mismo tablero en tiempo real. ✓
